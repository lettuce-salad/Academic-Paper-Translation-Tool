# 自動図表抽出ガイド

## 🎉 新機能：図表の自動抽出・埋め込み

PDFから図表を自動的に切り抜いて、HTMLに埋め込みます。

**図表の中は英語のまま** = 翻訳不要！

---

## 🚀 使い方

### 自動モード（推奨）

```cmd
python main.py paper.pdf --translate --page-mode
```

**自動的に実行されます**:
1. PDFを解析
2. 翻訳
3. **図表をPDFから切り抜き** ← NEW!
4. HTMLに埋め込み

---

## 📊 何が抽出されるか

### 1. 図（Figure）

PDFの図の部分を画像として切り抜き:

```
┌─────────────────────────────┐
│ Figure 1                    │
│ ┌─────────────────────────┐ │
│ │                         │ │
│ │ [PDFから切り抜いた     │ │
│ │  実際の図が表示]       │ │
│ │  (英語のまま)          │ │
│ │                         │ │
│ └─────────────────────────┘ │
│ 図1: ネットワーク構造       │
│     （キャプションは翻訳）  │
└─────────────────────────────┘
```

**特徴**:
- ✅ 元のPDFと同じ見た目
- ✅ 図の中は英語のまま（翻訳不要）
- ✅ キャプションだけ日本語に翻訳

---

### 2. 表（Table）

PDFの表の部分を画像として切り抜き:

```
┌─────────────────────────────┐
│ Table 1                     │
│ ┌─────────────────────────┐ │
│ │ A   │ B   │ C   │ D   │ │
│ │─────┼─────┼─────┼─────│ │
│ │ 1   │ 2   │ 3   │ 4   │ │
│ │ 5   │ 6   │ 7   │ 8   │ │
│ └─────────────────────────┘ │
│ 表1: 実験結果               │
│     （キャプションは翻訳）  │
└─────────────────────────────┘
```

**特徴**:
- ✅ 元のPDFと同じレイアウト
- ✅ 表の中は英語のまま
- ✅ キャプションだけ日本語に翻訳

## 🔧 仕組み

### ステップ1: 図表の位置を検出

PDFパーサーが図表の座標を取得:
```
Figure 1: (x0=100, y0=200, x1=500, y1=400)
Table 1: (x0=100, y0=500, x1=500, y1=700)
```

---

### ステップ2: 座標を指定して切り抜き

PyMuPDFで高解像度（150 DPI）で切り抜き:
```python
rect = fitz.Rect(x0, y0, x1, y1)
zoom = 150 / 72.0  # 高解像度
pix = page.get_pixmap(matrix=mat, clip=rect)
```

---

### ステップ3: Base64エンコード

PNG形式でBase64エンコード:
```python
img_data = pix.tobytes("png")
b64 = base64.b64encode(img_data).decode('utf-8')
data_uri = f"data:image/png;base64,{b64}"
```

---

### ステップ4: HTMLに埋め込み

```html
<div class="figure-container">
    <div class="figure-label">Figure 1</div>
    <img src="data:image/png;base64,iVBORw0KG..." 
         alt="Figure 1">
    <div class="figure-caption">図1: ネットワーク構造</div>
</div>
```

## 💡 Tips

### Tip 1: ファイルサイズ

Base64埋め込みなのでHTMLファイルが大きくなります:

```
図表なし: 500KB
図表15個: 5MB
```

**対処**: 問題ありません。現代のブラウザは問題なく表示できます。

## 🔍 トラブルシューティング

### 問題1: 図表が表示されない

**原因**: PDFパーサーが図表を検出していない

**確認**:
```cmd
# JSONで確認
cat paper_structure.json | grep -A 5 "figures"
```

**対処**: PDFの構造によっては検出できない場合があります

---

### 問題2: 図表の位置がずれる

**原因**: 座標の計算ミス

**対処**: 現在のバージョンではY座標順に配置されるので、ほぼ正確です

---

### 問題3: HTMLファイルが大きすぎる

**原因**: 図表が多い、または解像度が高い

**対処**:
1. 解像度を下げる（150 → 100 DPI）
2. 必要な図表だけ手動で選択（将来の実装）
