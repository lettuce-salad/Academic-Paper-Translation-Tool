# 図表処理ガイド

## 🎯 図表の扱い

PDFの図表は以下のように処理されます。

---

## 📊 現在の処理方式

### 1. 図（Figure）

**表示**:
```
┌─────────────────────────────┐
│ Figure 1                    │
│                             │
│    図 figure_001_0001       │
│    (画像はPDFから抽出)      │
│                             │
├─────────────────────────────┤
│ 図1: 性能比較                │
└─────────────────────────────┘
```

**特徴**:
- ✅ プレースホルダとして表示
- ✅ キャプションは翻訳される
- ✅ 元の位置（Y座標順）に配置
- ⚠️ 実際の画像は別途抽出が必要

---

### 2. 表（Table）

**表示**:
```
┌─────────────────────────────┐
│ Table 1                     │
├──────┬──────┬──────┬────────┤
│ A    │ B    │ C    │ D      │
├──────┼──────┼──────┼────────┤
│ 1    │ 2    │ 3    │ 4      │
│ 5    │ 6    │ 7    │ 8      │
├─────────────────────────────┤
│ 表1: 実験結果                │
└─────────────────────────────┘
```

**特徴**:
- ✅ HTMLテーブルとして表示
- ✅ キャプションは翻訳される
- ✅ 元の位置（Y座標順）に配置
- ✅ セル内のテキストも表示

## 🔧 図表が崩れる原因と解決

### 原因1: 順序が入れ替わる

**問題**:
```
段落1
段落2
図1 ← 本来はここ
段落3
図1 ← 実際はここに表示される
```

**解決**: Y座標でソート ✅

すでに実装済み。テキスト・図・表を統合してY座標順に並べます。

---

### 原因2: レイアウトが崩れる

**問題**:
```
テキストと図が重なる
表が画面からはみ出る
```

**解決**: 専用CSSスタイル ✅

```css
.figure-container {
    margin: 30px auto;  /* 上下に余白 */
    max-width: 90%;     /* 画面内に収める */
}

.table-container {
    overflow-x: auto;   /* 横スクロール可能 */
    max-width: 95%;     /* 画面内に収める */
}
```

## 🖼️ 実際の画像を埋め込む方法

### ステップ1: PDFから画像を抽出

```cmd
python src/parser/figure_extractor.py paper.pdf extracted_images
```

**出力**:
```
extracted_images/
  ├── page001_img000.png
  ├── page001_img001.png
  ├── page003_img000.png
  └── ...
```

---

### ステップ2: HTMLを編集

生成されたHTMLを開いて、プレースホルダを実際の画像に置き換え:

**修正前**:
```html
<div class="figure-placeholder">
    <svg>...</svg>
</div>
```

**修正後**:
```html
<div class="figure-placeholder">
    <img src="extracted_images/page003_img000.png" 
         alt="Figure 1" 
         style="max-width: 100%; height: auto;">
</div>
```

---

### ステップ3: 保存して確認

ブラウザでリロードすると実際の画像が表示されます。

## 💡 Tips

### Tip 1: 図が多い論文

プレースホルダのままでも読める場合が多いです。
必要な図だけ後で埋め込みましょう。

---

### Tip 2: 複雑な表

表のセルが多い場合、横スクロールで対応:

```css
.table-container {
    overflow-x: auto;  /* 自動で横スクロール */
}
```

---

### Tip 3: 印刷時

印刷時は図表が改ページされないように:

```css
.figure-container,
.table-container {
    page-break-inside: avoid;  /* ページ内に収める */
}
```

すでに実装済み ✅

## 🔍 トラブルシューティング

### 問題1: 図表が表示されない

**原因**: PDFパーサーが図表を検出していない

**確認**:
```cmd
# JSONで確認
cat paper_structure.json | grep -A 5 "figures"
```

**対処**: 現状ではプレースホルダのみ。画像抽出を手動で実行。

---

### 問題2: 表のセルがずれる

**原因**: セルの結合や複雑な構造

**対処**: HTMLを手動で調整

---

### 問題3: 図表とテキストが重なる

**原因**: Y座標が近すぎる

**対処**: CSSでマージンを調整
```css
.figure-container {
    margin: 40px auto;  /* 30px → 40px */
}
```
